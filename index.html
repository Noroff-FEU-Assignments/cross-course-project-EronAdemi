Web Security

How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

[The first thing the client should do is set up https certificates on the servers so that the data information is encrypted while transferring. Firewall rules and open ports should always be double checked. Important codes should render out from server side and not client side to prevent SQL injections or XSS ( Cross site scripting ). Using authentication headers. Validating sessions frequently. ]
